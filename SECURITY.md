# Security Policy

## Supported Versions

Currently, as iBooth is in its initial beta phases, only the latest active release branch is officially supported for security updates. 

| Version | Supported          |
| ------- | ------------------ |
| v0.0.x  | :white_check_mark: |
| Older   | :x:                |

## Reporting a Vulnerability

Security is a high priority for iBooth. We appreciate your efforts to responsibly disclose any vulnerabilities.

If you discover a security vulnerability within iBooth, please **do not** open a public issue on GitHub. Instead, please report it to the maintainers privately:

1. **Email:** Contact the maintainer directly (if you have their contact info) or use GitHub's private vulnerability reporting feature on the repository's "Security" tab.
2. **Details:** Please provide a detailed description of the issue, steps to reproduce it, and any potential impact.

We will acknowledge receipt of your vulnerability report as quickly as possible and strive to release a fix promptly.

Thank you for helping keep iBooth secure!
